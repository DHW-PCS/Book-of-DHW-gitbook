# 群聊和服务器发言管理

## \[界定]

一、刷屏行为

对于单一主体（群员、服务器玩家），1分钟内（除另有说明）发言超出以下容量之一者，界定有刷屏行为：

1. 加总150字符或8次换行；
2. 加总5表情、图片或链接；
3. 所有类型信息加总15条（或30秒内8条）。&#x20;

## \[处置]

视情节予以禁言处理，上限为14天。 对于机器人帐号，交互式消息在计数时将被视作使用者发出的消息。管理员得在处理前与其所有者联系（且/或协商）以采用其他解决方案。

## \[执行时间]

2022年2月12日起（总统已通过规定）。
