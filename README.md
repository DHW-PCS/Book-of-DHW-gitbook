# Book of DHW
## 简介
Book of DHW 是 DHW 官方通过 GitBook 公开本组资讯与规定的网站。为了方便本组成员及 DHW Inf 玩家等相关人士编辑及使用本文档，本组已在 2022 年 12 月 9 日将该网站迁移到 GitHub。
